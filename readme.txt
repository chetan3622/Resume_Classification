Resume Classifier Project

How to run this project:

1. Open terminal inside this folder
2. Install required packages:
   pip install -r requirements.txt

3. If model files are missing, run:
   python train_model.py

4. Start the app:
   streamlit run app.py

Files included:
- app.py
- train_model.py
- resume_dataset.csv
- model.pkl
- tfidf.pkl
- label_encoder.pkl
- requirements.txt
- README.txt
